# dent-

[link](https://www.etymonline.com/word/*dent-#etymonline_v_52695)

Proto-Indo-European root meaning "tooth."

Examples:

1. dent
2. dent = dent + al
3. dentist = dent + ist
4. denture = dent + ure
5. dandelion = dan + de (...of) + lion = the teeth of the lion
6. dentifrice = dent + ti +frice
7. indent = in + dent
